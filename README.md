# CodeCollab 🚀

A real-time collaborative code editor with isolated rooms.

## 🛠️ Tech Stack
- **Frontend:** React, Monaco Editor, Vite
- **Backend:** Node.js, WebSockets
- **Sync:** Yjs, y-websocket

## 🚀 How to Run

### 1. Start the Server
cd server
npm install
node start-sync.cjs

### 2. Start the Client
cd client
npm install
npm run dev
